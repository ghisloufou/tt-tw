## React Game Of Thrones Bookshelf

Objective was from a technical test to build a Game of thrones bookshelf based on the following api:
https://anapioficeandfire.com/

### Start the app

run:

``` shell
npm install
```

then

``` shell
npm run start
```

or with pnpm:  

``` shell
pnpm install
```


then

``` shell
pnpm start
```
